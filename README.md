# MDM Review Portal

One GitHub repo, two portals — all files live at the root.

## URLs (after deploying to GitHub Pages)
- Blog Review:   https://thecreatorie.github.io/[repo-name]/
- Social Review: https://thecreatorie.github.io/[repo-name]/social-index.html

## Files
- `index.html` + `post-*.html` + `style.css` + `app.js`  → Blog portal
- `social-index.html` + `social-post-*.html` + `social-style.css` + `social-app.js` + `social-posts-data.js`  → Social portal

## Deploy
1. Create a GitHub repo (e.g. `mdm-review`)
2. Upload ALL files to the root
3. Settings → Pages → Source: Deploy from branch → main → / (root)

# MDM Design Studio — Blog Review Portal

Private client review portal for the 2026 blog publishing schedule.

## Live site
Once GitHub Pages is enabled, the portal will be live at:
`https://[your-username].github.io/[repo-name]/`

## How to enable GitHub Pages
1. Go to this repository's **Settings**
2. Click **Pages** in the left sidebar
3. Under **Source**, select **main** branch and click **Save**
4. Your site will be live within ~60 seconds at the URL shown

## Files
- `index.html` — dashboard with all 25 posts
- `posts/` — individual post review pages
- `app.js` — live sync logic (Firebase + EmailJS already configured)
- `style.css` — shared styles
